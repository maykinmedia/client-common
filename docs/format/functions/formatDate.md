# Function: formatDate()

> **formatDate**(`date`, `format`): `string`

Formats a `Date` into a string.

## Parameters

### date

`Date`

The date to format.

### format

The output format.

Supported formats:
- `"iso"` (default) — `YYYY-MM-DD`
- `"nl"` — `DD/MM/YYYY`

`"iso"` | `"nl"`

## Returns

`string`

The formatted date string.

## Throws

Error If an unsupported format is provided.
