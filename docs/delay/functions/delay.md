# Function: delay()

> **delay**(`timeout`): `Promise`\<`void`\>

Returns a `Promise` which resolves after the specified timeout.

## Parameters

### timeout

`number` = `300`

Time in milliseconds to wait before resolving. Defaults to 300.

## Returns

`Promise`\<`void`\>
